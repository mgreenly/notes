Software Development and Data Analysis for Small Businesses

About

Logic Refinery is a software development consultancy that focuses on the needs of small and medium sized businesses.

It was formed in 2010 by Michael Greenly, owner and principal developer, and operates in east centeral Minnesota.

Services

Logic Refinery provides custom software development and data analysis for small businesses.

This commonly involves the development of custom web applications that interface with backend ERP or bookkeeping systems to provide deeper insights into business performance. As well as customer portals, online stores, data collection tools and automated report generation.
